# 📈 大漲訊號自動選股系統

依林則行《大漲的訊號》「創新高價股」投資法打造的全自動雲端選股系統。

- **掃描範圍**：台股上市＋上櫃全部（約 1800 檔）、美股 S&P 500 + NASDAQ 100（約 600 檔）
- **買進訊號**：突破 2 年新高 ＋ 9 項檢核表評分卡（反彈幅度、長短期獲利成長、營收動能、本益比、大盤力道）
- **賣出監控**：8% 停損、單季獲利年增 <20%、賣壓比例 SPR ≥117%
- **通知**：Gmail 自動寄送買賣訊號
- **儀表板**：GitHub Pages 網頁，紅黃綠燈＋評分卡，新手可讀
- **全自動**：GitHub Actions 每日排程（台股 16:30、美股 06:00 台北時間），電腦關機照跑，不需任何 AI 訂閱，全免費

## 快速開始

看 [SETUP.md](SETUP.md) 一步步設定（約 30 分鐘）。

## 專案結構

```
engine/            選股引擎（純 Python）
  spr.py           賣壓比例（書第四章第五節）
  signals/         突破偵測、基本面檢核、大盤燈號、賣出監控
  scoring.py       9 項檢核表評分卡（書附錄一）
  run_daily.py     每日執行入口
tests/             單元測試（含書中數值範例重現）
docs/index.html    儀表板（自動產生）
holdings.csv       你的持股（自己編輯）
config.yaml        門檻設定（皆出自書中）
.github/workflows  每日自動排程
```

## 本機測試

```
pip install -r requirements.txt
python -m pytest tests/           # 跑測試
python -m engine.run_daily --market tw --no-email --limit 100   # 小樣本試跑
```
