新手三分鐘看懂這套方法

1. 只買創新高的股票：突破 2~3 年高價代表公司進入新時代，之前要有長而平穩的整理期（能量累積）。
2. 基本面要加速：長期獲利年均 ≥7%、近年 ≥20%、近幾季營收 ≥10% 且獲利 ≥20%，本益比 <60 倍。

4. 看大盤臉色：創新高股愈多行情愈強；紅燈時系統自然找不到訊號，空手就是策略。
5. 資金控管：單檔不超過總資產 10%，行情弱就再減量。
